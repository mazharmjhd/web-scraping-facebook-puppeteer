PUPPEETER Puppeteer adalah perpustakaan Node yang menyediakan API tingkat tinggi untuk mengontrol Chrome atau Chromium atas Protokol DevTools. Dalang berjalan tanpa kepala secara default, tetapi dapat dikonfigurasikan untuk menjalankan Chrome atau Chromium penuh (tanpa-kepala). Apa yang dapat saya? Sebagian besar hal yang dapat Anda lakukan secara manual di peramban dapat dilakukan menggunakan Puppeteer! Berikut adalah beberapa contoh untuk membantu Anda memulai: Hasilkan tangkapan layar dan PDF halaman. Merayapi SPA (Aplikasi Satu Halaman) dan menghasilkan konten yang dirender sebelumnya (mis. "SSR" (Rendering Sisi Server)). Mengotomatiskan pengiriman formulir, pengujian UI, input keyboard, dll. Buat lingkungan pengujian otomatis yang mutakhir. Jalankan pengujian Anda langsung di versi terbaru Chrome menggunakan fitur JavaScript dan browser terbaru. Tangkap jejak garis waktu situs Anda untuk membantu mendiagnosis masalah kinerja. Uji Ekstensi Chrome. Beri putaran: https://try-puppeteer.appspot.com/
